The failure to achieve stable upright balancing is directly attributable to the dataset description's constraint: "The native Gymnasium reward is replaced entirely by the Lyapunov-based reward." 

As noted in the analysis, this constraint transforms the reward into a potential-based function where the cumulative return is a telescoping sum dependent only on the initial and final states. This eliminates the dense, continuous penalty required to incentivize the agent to maintain the upright position against the entropy-maximizing objective of the SAC algorithm. The observed "stabilization paradox" is a direct consequence of this specific reward curation choice.
