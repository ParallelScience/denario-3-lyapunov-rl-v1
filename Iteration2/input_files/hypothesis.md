The previous iterations suggest that while the Lyapunov prior $\Phi(s)$ provides a strong physical scaffold for the value function, it acts as a "gradient trap" when fixed, as it cannot account for the agent's specific control constraints or the discount factor $\gamma$. I hypothesize that replacing the static Lyapunov prior with a **Gated Lyapunov-Residual Architecture** will improve performance. 

Instead of $V(s) = \Phi(s) + f_\theta(s)$, we define $V(s) = \alpha(s) \cdot \Phi(s) + f_\theta(s)$, where $\alpha(s)$ is a small, learnable gating network (or a scalar parameter initialized to 1) that allows the model to dynamically attenuate or amplify the influence of the Lyapunov prior based on the state. 

Furthermore, to address the "telescoping reward" issue identified in Iteration 0, we will implement **Potential-Based Reward Shaping (PBRS)**: $R'_t = R_{native} + \gamma \Phi(s_{t+1}) - \Phi(s_t)$. This preserves the optimal policy of the original Pendulum-v1 environment while using the Lyapunov function to provide a dense, informative gradient. By combining the gated structural prior with PBRS, the agent will retain the physical guidance of $\Phi(s)$ while gaining the flexibility to ignore the prior in regions of the state space where the Lyapunov function is a poor proxy for the true value, thereby resolving the optimization instability observed in PPO.
