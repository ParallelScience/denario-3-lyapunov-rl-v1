The current structured value function $V(s) = \Phi(s) + f_\theta(s)$ effectively captures the global energy landscape, but the "pure" Lyapunov reward fails to incentivize precise stabilization because it treats the goal state as a zero-reward sink rather than a high-value equilibrium. I hypothesize that by combining the structured critic with Potential-Based Reward Shaping (PBRS) — specifically using $R_t = R_{native} + \gamma \Phi(s_{t+1}) - \Phi(s_t)$ — we can preserve the optimal policy of the native environment while using $\Phi(s)$ to provide a dense, stable gradient. Furthermore, I hypothesize that the residual network $f_\theta(s)$ in this configuration will specifically learn to compensate for the "energy-dissipation" gap between the uncontrolled pendulum and the torque-limited agent, leading to superior upright stability compared to both the pure Lyapunov reward and standard SAC.
